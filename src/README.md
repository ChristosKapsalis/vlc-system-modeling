# Source Notebooks

This folder contains the computational analysis notebooks used for VLC system modeling and experimental validation.

## Notebooks

- `Lambertian_Model.ipynb`  
  Simulates Lambertian radiation patterns for different mode numbers.

- `Heatmap_Generation.ipynb`  
  Generates optical intensity heatmaps for LOS VLC transmitter configurations.

- `Theoretical_VS_Experimental.ipynb`  
  Compares theoretical Lambertian model predictions with experimental measurements.

- `least_squares_fit.ipynb`  
  Applies least-squares fitting for model validation.

- `Arduino_Brightness_Analysis.ipynb`  
  Analyzes brightness measurements acquired using Arduino-based receiver setups.

- `Voltage_Distance_Analysis.ipynb`  
  Studies voltage/intensity variation with transmitter–receiver distance.

- `Multi_Distance_Analysis.ipynb`  
  Evaluates optical signal behavior across multiple measurement distances.
